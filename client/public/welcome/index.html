Your First Steps
Below is a suggested 3 day course where we encourage hands-on development with guided steps on how to build and run Genesis applications. You'll cover the configuration of multiple Genesis components with the ultimate goal of building an FX Trading platform.

Day 1:
- Introduce the Genesis workflow with 20 minute introductory video showing how quickly an application can be created
- Start the development journey with Genesis Create - using the First Project Guide we'll guide you through building the FX Trading platform, configuring the datamodel and stepping you through the component configuration
- Learn more about the structure of a Genesis application and the tooling used in a development environment
- Generate your project and deploy services using the Genesis Intellij plugin

Day 2:
- Discover more about the Genesis platform with in-depth, academy training
- Review the components you configured on day 1 for a more detailed understanding of their roles in a Genesis environment
- Finish the day with a certificate that completes your Genesis training!

Day 3:
- Revisit the trading application built on day 1
- Configure notifications and alerts as well as adhoc reporting
- Improve consolidator logic by enhancing events
- Augment the UI as we guide you through the steps to pivot your data
